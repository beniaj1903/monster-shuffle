# PokeRandomWeb Frontend

Frontend React + TypeScript + Vite para PokeRandomWeb.

## Instalación

```bash
cd web
npm install
```

## Desarrollo

```bash
npm run dev
```

El servidor de desarrollo se iniciará en `http://localhost:5173`.

## Build

```bash
npm run build
```

## Características

- ✅ Cliente API completo (`src/api.ts`)
- ✅ Máquina de estados visual para el flujo de juego
- ✅ Sincronización automática con el servidor
- ✅ Interfaz responsive con CSS moderno

## Flujo de Juego

1. **Inicio**: Click en "Nuevo Juego" para crear una sesión
2. **Starter Selection**: Elige uno de los 3 starters disponibles
3. **Mapa**: Ve tu equipo y explora la hierba alta
4. **Encuentro**: Elige una de las 5 opciones a ciegas
5. **Revelación**: Descubre qué Pokémon capturaste
